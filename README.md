# Party Game (Python)

Ein kleines Python-Spiel mit zufälligen Aufgaben für mehrere Spieler.

## Funktionen
- Spieler können hinzugefügt werden
- Zufällige Aufgaben werden ausgewählt
- Einfache Konsolensteuerung

## Technologien
- Python
- random Modul

## Status
Projekt zum Lernen und Üben von Python-Grundlagen
